Is It Localhost ?

Configuration

Aide

Afin que l'extension fonctionne correctement, veuillez vérifier les points suivants :

1. Création d'un vhost
Vous devez créer un vhost avec le nom de domaine : is-it-localhost.local
Ce vhost doit pointer vers le dossier où se trouve l'extension
Exemple de vhost : 
              <VirtualHost *:80>
                  ServerAdmin example@email.com
                  DocumentRoot "/web/root/is-it-localhost/"
                  ServerName is-it-localhost.local
              </VirtualHost>
2. Permissions et groupe
Vous devez vérifier les permissions sur les fichiers et dossiers.
Si vous êtes sur un serveur Apache, habituellement le groupe est "www-data"
